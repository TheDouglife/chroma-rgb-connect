# Chroma RGB Connect
Chroma Connector App for other 3rd party RGB LED providers.

## Getting Started
This repository contains the source code for the primary Chroma RGB Connect installable components. Development and maintenance now live in this repository at [TheDouglife/chroma-rgb-connect](https://github.com/TheDouglife/chroma-rgb-connect).

## Project Continuation
This repository is the maintained continuation of the original Chroma Control project. The codebase, issue tracking, releases, and branding now live here under the Chroma RGB Connect name.

## Download
Downloads for Chroma RGB Connect can be found on the [releases page](https://github.com/TheDouglife/chroma-rgb-connect/releases) and the [installer page on GitHub Pages](https://thedouglife.github.io/chroma-rgb-connect/installer/).

## Reporting security issues and bugs
You can follow the instructions in our [Security Policy](https://github.com/TheDouglife/chroma-rgb-connect/security/policy) to report security related issues. You can also find the full list of supported Chroma RGB Connect versions there as well.
